# شجرة الأوامر - Command Tree

## 🔐 إدارة وإشراف
- `!سوول @عضو` - إعطاء صلاحيات مسوول كاملة
- `!مشرف @عضو` - إعطاء صلاحيات مشرف
- `!صلاحياتي` - عرض صلاحياتك

## 👮 إدارة الأعضاء
- `!kick @عضو` - طرد عضو
- `!ban @عضو` - حظر عضو
- `!unban <id>` - فك الحظر
- `!timeout @عضو <وقت>` - كتم عضو
- `!untimeout @عضو` - فك الكتم
- `!اعطا_رتبة @عضو @رتبه` - إعطاء رتبة
- `!سحب_رتبة @عضو @رتبه` - سحب رتبة

## 📁 إدارة الرومات
- `!انشاء_روم <اسم>` - إنشاء روم نصي
- `!حذف_روم` - حذف الروم الحالي
- `!حذف_روم_صوتي <اسم>` - حذف روم صوتي
- `!حذف_تيم <اسم>` - حذف تيم كامل
- `!تسمية_روم <اسم>` - تغيير اسم الروم
- `!اسم_السيرفر <اسم>` - تغيير اسم السيرفر
- `!وصف_السيرفر <نص>` - تغيير الوصف
- `!صورة_السيرفر <رابط>` - تغيير الصورة
- `!قفل` - قفل الروم
- `!فتح` - فتح الروم
- `!بطيء <ثواني>` - وضع بطيء

## 🎭 إدارة الرتب
- `!انشاء_رتبة <اسم>` - إنشاء رتبة
- `!لون_رتبة @رتبه <لون>` - تغيير لون الرتبة
- `!اعادة_تسمية_رتبة @رتبه <اسم>` - إعادة تسمية الرتبة
- `!رتب_ترحيب @رتبه` - رتبة ترحيب تلقائية
- `!الغاء_رتب_ترحيب` - إلغاء رتبة الترحيب

## 📝 معلومات
- `!معلومات @عضو` - معلومات العضو
- `!سيرفر` - معلومات السيرفر
- `!صورة @عضو` - صورة الأفاتار
- `!سحب <id>` - البحث عن عضو
- `!ping` - سرعة البوت

## 💬 أدوات
- `!say <نص>` - قول شيء
- `!clear <عدد>` - مسح رسائل
- `!snipe` - رسالة محذوفة
- `!poll <سؤال>` - تصويت
- `!vote` - تصويت ( Slash )

## 🎮 الألعاب (لعبة)
- `!لعبة حجر` - حجر ورقة مقص
- `!لعبة تخمين` - تخمين رقم
- `!لعبة روليت` - روليتロシア
- `!لعبة كازينو` - كازينو
- `!لعبة نرد` - نرد
- `!لعبة عملة` - عملة
- `!لعبة اكس او @عضو` - اكس او
- `!تحدي_عاز` - تحدي عاز

## 🛡️ الحماية (حماية)
- `!حماية سبام` - منع السبام
- `!حماية فلود` - منع الفلود
- `!حماية منشن` - منع المنشن الجماعي
- `!حماية كلمات` - فلترة الكلمات
- `!حماية انفايت` - منع الروابط
- `!حماية الت` - منع الحسابات الجديدة
- `!حماية ريد` - منع الرابد
- `!حماية تهكير` - Anti-Nuke (حذف جماعي للرومات/الرتب)
- `!حماية الكل` - تشغيل كل الحماية
- `!حماية حالة` - عرض حالة الحماية
- `!حماية كلمات_اضافة <كلمة>` - إضافة كلمة
- `!حماية كلمات_حذف <كلمة>` - حذف كلمة
- `!حماية استثناء #قناة <نوع>` - استثناء قناة من نوع حماية

## 🔗 اللوق (log) — 12 روم
- `!log setup` - إعداد اللوق (ينشئ 12 روم)
- `!log set <نوع> #روم` - تعيين قناة
- `!log status` - عرض حالة الرومات
- `!log reset` - إعادة تعيين
- `!log delete` - حذف الرومات
